# Security

pdf-merger runs entirely in your browser — nothing is uploaded, so there is no server to attack. The client code and its dependencies still matter.

Report vulnerabilities privately, **not** via a public issue:

- [GitHub Security Advisory](https://github.com/dlepaux/pdf-merger/security/advisories/new) (preferred)
- Email `d.lepaux@gmail.com`, subject `[security][pdf-merger]`

Acknowledged within 72h. Coordinated disclosure; no public detail until a fix ships.
